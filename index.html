About Me

Berada pada semester 5 sebagai mahasiswa, saya tidak hanya membawa semangat belajar, tetapi juga visi besar untuk masa depan di bidang teknologi dan bisnis. Menjadi Developer profesional dan seorang pengusaha visioner adalah dua mimpi besar yang saya tuju, dan setiap langkah yang saya ambil dirancang untuk membawa saya lebih dekat dengan tujuan tersebut.

Perjalanan saya dalam dunia teknologi telah memandu saya untuk mengeksplorasi kedalaman Java dan PHP. Saya memiliki pengalaman praktek dengan framework CI (CodeIgniter) dan Laravel, yang keduanya telah meningkatkan kemampuan dasar saya dan memperluas pemahaman saya tentang dunia pengembangan.

Dengan mengunjungi portofolio ini, Anda tidak hanya akan melihat kemampuan teknis saya tetapi juga semangat, dedikasi, dan ambisi saya untuk meningkatkan dan berinovasi. Saya yakin bahwa kombinasi antara kemampuan teknis dengan semangat kewirausahaan saya akan membawa perspektif unik ke setiap proyek atau perusahaan.
